## рџљЂ HOW TO USE THIS SYSTEM

### Daily Workflow
1. **Start your day**: Check `[EXECUTIVE_DASHBOARD.md](dashboards/EXECUTIVE_DASHBOARD.md)` for today's priorities
2. **Update status**: Edit `[today.md](dashboards/today.md)` throughout the day
3. **Log activity**: Update relevant project or BD files
4. **Review tonight**: Check if anything needs follow-up

### Weekly Workflow
1. **Monday morning**: Review `[weekly-review.md](dashboards/weekly-review.md)`
2. **Update KPIs**: Refresh `[weekly-metrics.md](analytics/weekly-metrics.md)`
3. **Plan next week**: Set priorities in Executive Dashboard
4. **Friday afternoon**: Archive completed items, prep for next week

### Project Management
- Each project folder follows the same structure (objectives, deliverables, action items, progress)
- Use project READMEs as your single source of truth
- Update progress.md regularly

### Business Development Pipeline
- Track all prospects in `[01-prospects-leads.md](business-development/01-prospects-leads.md)`
- Manage proposals in `[02-proposals.md](business-development/02-proposals.md)`
- Log all client communications in `[03-client-communications.md](business-development/03-client-communications.md)`
